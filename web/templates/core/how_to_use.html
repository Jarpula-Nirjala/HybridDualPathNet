How to Use

Follow this step-by-step guide to upload brain MRI scans and get AI-powered disease classification with explainable heatmaps.

1

Create an Account

Sign up with a username and password. Your scan history is saved securely to your account.

{% if not user.is_authenticated %} Sign Up Now {% else %} You're logged in as {{ user.username }} {% endif %}
2

Upload a Brain MRI Image

Go to Upload and select a brain MRI image. Supported formats: JPG, PNG. Maximum file size: 10 MB. You'll see a live preview before submitting.

3

Wait for AI Analysis

The processing page polls every 2 seconds. Our hybrid model runs 7-pass Test-Time Augmentation (TTA) inference and generates a Grad-CAM++ heatmap. On CPU this typically takes 30–90 seconds.

4

Review Results

View the predicted class (AD / PD / Control), confidence bar chart, side-by-side original MRI vs heatmap overlay, and processing time. Results are saved to your history.

5

Browse Scan History

Access History to see all past scans with thumbnails, status badges, and quick links to re-view any result.

Supported Classes

AD — Alzheimer's Disease PD — Parkinson's Disease CONTROL — Healthy

  • Model weights must be present in weights/ for inference to succeed.
